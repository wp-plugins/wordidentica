WordIdentica is a plugin that creates a complete integration between your WordPress blog and your Identi.ca account.

== Description ==

WordIdentica integrates with Identi.ca by giving you the following functionality of creating a post on Identi.ca whenever you post in your blog, with a link to the blog post.



== Installation ==

1. Download the plugin archive and expand it (you've likely already done this).
2. Put the 'wordidentica.php' file into your wp-content/plugins directory.
3. Go to the Plugins page in your WordPress Administration area and click 'Activate' for WordIdentica.
4. Go to the WordIdentica Options page (Options > wordidentica) to set your Identi.ca account information.


== Configuration ==

There are a no configurations for WordIdentica at this time.



== Known Issues ==

* Only one Identi.ca account is supported (not one account per author).
* Posts are not deleted from the identi.ca table in your WordPress database when they are deleted from identi.ca. To delete from your WordPress database, use a database admin tool like phpMyAdmin.
* The relative date function isn't fully localized.
